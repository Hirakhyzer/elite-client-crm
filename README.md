# Elite Client CRM

A polished white, black, and `#f4af00` gold React CRM dashboard for managing clients, sales opportunities, pipeline stages, invoices, calendar events, emails, goals, client tasks, and activity logs.

Built for **Elite Era Development L.L.C**.

## Features

- White and gold Elite Era Development design system
- Responsive sidebar navigation
- Dashboard metrics and pipeline funnel
- Client search, filters, sorting, and detailed profiles
- Add, edit, and delete clients
- Lead score, tags, tier, industry, deal value, and pipeline fields
- Client tasks and activity timeline
- Kanban-style sales pipeline with stage updates
- Calendar events
- Invoices with editable payment status
- Email draft history
- Business goals with progress tracking
- Browser local storage persistence
- Mobile responsive layout
- Footer: Made by Hira Khyzer

## Tech Stack

- React
- Vite
- JavaScript
- CSS

## Run Locally

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal.

## Build for Production

```bash
npm run build
```
